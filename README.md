🧬 What is Phylogenix?

Phylogenix is an interactive modular cladogram platform that helps users explore evolutionary relationships, trace common ancestry, identify closest relatives, and visualize life's history through geological time.

Each organism includes a dedicated profile card containing key biological and taxonomic information.

Designed for education, research, and science communication.

---

✨ Features

- 🌳 Interactive phylogenetic cladogram
- 🔍 Instant taxon search and navigation
- 📅 Geological timeline integration
- 🧬 Detailed organism profile cards
- 🖼️ Taxon image support
- 📖 Taxonomic classification viewer
- ↔️ Pan and zoom controls
- ⚙️ Developer Mode for cladogram editing
- ↩️ Undo / Redo functionality
- 💾 Export as standalone HTML
- 📱 Responsive design

⚠️ Important Note

This project requires a backend image-hosting service for loading organism images and media assets.

The current implementation utilizes Supabase Storage.

Without a configured backend service, all cladogram and timeline functionality will remain operational, but image assets may not load correctly.

Developer Warning

The Developer Mode (⚙️) and Spawn Mode tools are intended for developers only.

Avoid clicking, long-pressing, or modifying these controls unless you understand the underlying data structure. Improper use may alter lineage relationships, disrupt cladogram organization, or damage project data.

For deployment assistance or backend configuration, please contact the author.
