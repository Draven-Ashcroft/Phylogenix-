User Manual v1.0

EvoGen — Mapper

Interactive cladogram simulation for mapping vertebrate evolutionary lineages with matrix precision.

1

Getting Started

📂
Open the File
Double-click the .html file to open it in any modern browser (Chrome, Edge, Brave, Firefox). No server or installation needed — it runs completely offline.
🧬
Choose a Clade
The tab bar at the top shows available clades — Sauropsida Clade and Synapsida Clade. Click a tab to switch between the two evolutionary lineages. Both share the same geological timeline.
🔍
Search Organisms
Use the search bar at the top to quickly find any organism across both clades. Type a name and the matching node will be highlighted on the canvas.

3

Viewing Profile Cards

👆
Select a Node
Click on any organism node (the coloured circles) on the cladogram to select it. The node name appears in the bottom panel.
🪪
Open Profile Card
Click the PROFILE CARD button in the top-right to view the selected organism's detailed card. The card shows scientific name, image, chrono horizon, ancestor, diet, habitat, common name, and distinguishing characters.
⬅️
Return to Cladogram
Click ← BACK at the top of the profile card to return to the cladogram view. You can also click any clade tab to return.

Profile Card — Field Guide

FieldDescription
Scientific NameBinomial nomenclature or clade name of the organism
Taxonomic ClassificationHierarchical position — Clade, Order, Family
Status🟢 Living or ❌ Extinct — with extinction date if applicable
ImageRepresentative specimen — real photograph or fossil-based AI reconstruction
Chrono HorizonTimeline bar showing when the organism arose (green) and went extinct (red) or Present if living
AncestorAuto-computed from the cladogram tree structure — shows the parent node
DietFeeding ecology — Carnivore, Herbivore, Omnivore, Insectivore, Piscivore etc.

HabitatEcological niche — Terrestrial, Aquatic, Marine, Aerial etc.
Distinguishing CharactersKey morphological or behavioural features that define this group

Editing Profile Cards

✏️
Enter Edit Mode
With a profile card open, click the EDIT button. All fields become editable — click on any field to type or modify its content.
🖼️
Set Profile Image
In edit mode, the image URL input appears below the image. Enter the Supabase path (e.g. Dinosauria/Trex.webp) and click SET. Images are hosted on Supabase for reliable loading.
💾
Save Changes
Click DONE EDITING to save your changes. All modifications are saved to the in-memory data — remember to export a snapshot to preserve them permanently.
⚠️
Important: Edits are saved in browser memory only. If you close the tab without exporting a snapshot, all changes are lost. A browser warning will appear, but always export after editing.
6

Evolution Track

📊
Geological Timeline
The horizontal axis represents geological time in Millions of Years Ago (Mya) — from the Late Carboniferous (315 Mya) to Present (0 Mya). Period labels (Triassic, Jurassic, Cretaceous etc.) are displayed along the timeline.
🌿
Reading the Tree
Each node represents an organism or clade. Branches connect ancestors to descendants. The horizontal position shows when the group arose. Living groups (🟢) have branches extending to Present. Extinct groups (†) have branches that stop at their extinction point.
🔀
Dual Clade View
Switch between Sauropsida (reptiles and birds) and Synapsida (mammals) clades using the tab buttons. Both share the common ancestor "Early Amniotes" at the root.
7

Exporting & Saving Your Work

Every export creates a complete, self-contained HTML file with all your data baked in. Each snapshot is a permanent save point.

Finish Editing
Click DONE EDITING if you're in edit mode. Ensure all changes are saved.
Close Profile Card
Click ← BACK to return to the cladogram view. Close any open profile tabs.
Lock Controls
Make sure pan/zoom is locked (lock icon should show locked state).
Export Snapshot
Click EXPORT HTML SNAPSHOT in the control panel. A new .html file is downloaded — this is your saved version.
Verify
Open the exported file in a new tab to confirm all data, images, and node positions are preserved correctly.
⚠️
Save often: Export a snapshot after every 2–3 nodes you fill. Each snapshot is a version checkpoint — if anything goes wrong, you can always go back to the previous one.
8

Developer Mode (Advanced)

For advanced editing of the cladogram structure. Long-press the gear icon (⚙️) for 2 seconds to activate.

⚙️
Activate Dev Mode
Long-press the gear icon in the top-right. The gear glows cyan when active. The Developer Panel opens on the left with the full JSON data matrix.
🔧
JSON Data Matrix
View and edit the raw node/edge data. You can modify node positions (x, y), labels, MYA values, and profile data. Click APPLY CHANGES to save (a confirmation dialog will appear).
🌱
Spawn Mode
Click the sprout icon to enter Spawn Mode for adding new nodes and edges to the cladogram. Click on existing nodes to branch from them.
↩️
Undo / Redo
Use Ctrl+Z to undo and Ctrl+Y to redo structural changes. Works across both clades.
⚠️
Caution: Developer Mode gives full access to the data structure. Incorrect JSON edits can break the cladogram. Always export a snapshot before making structural changes.

Troubleshooting

ProblemSolution
Image not loadingCheck the Supabase path is correct. Ensure you have internet connectivity — images are hosted externally.
Can't pan or zoomClick the lock icon to unlock pan/zoom first. The lock must be in the unlocked state.
Profile card shows zoom controlsThis shouldn't happen in the latest version. If it does, close the profile card and reopen it.
Text cropped in diet/habitat fieldsFields auto-expand in the latest version. If cropped, you may be on an older snapshot — re-export from the latest base file.
Lost my editsIf you didn't export before closing, the data is lost. Always export snapshots after editing. Open your most recent snapshot to continue.
Node label clipped at edgeAdjust the node's Label X/Y Offset in Spawn Mode, or pan the canvas slightly.
